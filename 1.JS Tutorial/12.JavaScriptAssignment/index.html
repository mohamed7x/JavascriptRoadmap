JavaScript Assignment Operators

Assignment operators assign values to JavaScript variables.

Operator Example Same As
= x=y y=x
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
<<= x <<= y x = x << y
>>= x >>= y x = x >> y
>>>= x >>>= y x = x >>> y
&= x &= y x = x & y
|= x |= y x = x | y
^= x ^= y x = x ^ y
**= x **= y x = x ** y
